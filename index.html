Minha Pesquisa sobre HTML e CSS

O que é HTML?

O HTML é uma linguagem usada para estruturar o conteúdo de páginas da web.

O que é CSS?

O CSS é usado para estilizar a página (cores, fontes, tamanhos), enquanto o HTML é usado para organizar o conteúdo.

Sites ou aplicativos que usam HTML/CSS

Exemplo 1: YouTube
Exemplo 2: Facebook, Google, Instagram, etc.

O que é responsividade?

Responsividade significa quando um site se adapta a diferentes tamanhos de tela, como celular, tablet e computador.

Importância do GitHub

O GitHub é importante porque permite salvar, compartilhar e colaborar em projetos de programação.
